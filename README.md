ExpenseAI – Smart Student Expense Tracker
ExpenseAI is a responsive, modern frontend web application designed to help students manage, analyze, and control their monthly expenses with intelligent analytics and user-friendly dashboards.

🚀 Features
User Authentication: Secure registration, login, and data privacy.

Intuitive Dashboard: Visual overview of total spend, budget left, and analytics.

Expense Management: Add, edit, delete, and search expenses by category, amount, or date.

Budget Tracking: Set monthly budgets, get notifications on overspending, and category-wise analysis.

AI-powered Insights: Get smart recommendations and warnings for your spending habits.

SMS Parsing: Automatically detect and categorize transaction SMS.

Charts & Analytics: Rich visualizations for trends, heatmaps, period-wise comparison using Chart.js.

Customizable UI: Light/dark modes, currency, language, and notification settings.

Data Export: Download your transaction history as CSV.

🛠️ Tech Stack
HTML5 + CSS3 (responsive with Bootstrap)

JavaScript (ES6+)

Chart.js for data visualizations

FontAwesome for icons

LocalStorage (for session/data persistence; replaceable by backend)
